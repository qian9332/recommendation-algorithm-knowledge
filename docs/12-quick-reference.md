## 12.1 核心公式速查

### 召回层

| 公式名称 | 公式 | 说明 |
|---------|------|------|
| **BM25** | `Score = Σ IDF(t) × [tf·(k1+1)] / [tf+k1·(1-b+b·\|D\|/avgdl)]` | k1=1.2~2.0, b=0.75 |
| **InfoNCE** | `L = -log(exp(s⁺/τ) / Σ exp(s/τ))` | τ=0.05~0.1 |
| **余弦相似度** | `sim = u·v / (\|u\|·\|v\|)` | 向量召回 |

### 排序层

| 公式名称 | 公式 | 说明 |
|---------|------|------|
| **FM二阶项** | `y = Σ⟨v_i, v_j⟩x_i x_j` | O(kn)复杂度 |
| **DeepFM交叉** | `y = y_FM + y_DNN` | 结合显式+隐式 |
| **注意力分数** | `α = softmax(QK^T/√d)` | Transformer |

### 多任务学习

| 公式名称 | 公式 | 说明 |
|---------|------|------|
| **门控输出** | `g = softmax(z/τ)` | τ控制平滑度 |
| **GradNorm** | `L_grad = Σ\|\|G_i\|\| - α·r_i^β\|` | 梯度平衡 |
| **不确定性加权** | `L = Σ(1/2σ²)·L_i + logσ` | 自动权重 |
| **ESMM** | `pCTCVR = pCTR × pCVR` | 全空间建模 |

### 重排层

| 公式名称 | 公式 | 说明 |
|---------|------|------|
| **DPP核矩阵** | `L_ij = q_i · q_j · S_ij` | 质量×相似度 |
| **疲劳衰减** | `w(k) = exp(-λk)` | λ=0.3~0.5 |
| **位置折扣** | `w(pos) = 1/log(1+pos)` | NDCG权重 |

### 强化学习

| 公式名称 | 公式 | 说明 |
|---------|------|------|
| **策略梯度** | `∇J = E[∇logπ(a\|s)·R]` | REINFORCE |

| **IPS估计** | `V̂ = (1/n)Σ(π/β)·r` | Off-policy |

3. **序列特征如何建模？**

1. **MMoE与PLE区别？**

2. **跷跷板现象如何缓解？**

3. **ESMM解决了什么问题？**

1. **DPP如何保证多样性？**

## 12.7 代码模板速查

### TensorFlow多任务模型

```python
class MultiTaskModel(tf.keras.Model):
    def __init__(self):
        super().__init__()
        self.embedding = tf.keras.layers.Embedding(vocab_size, dim)
        self.ctr_tower = tf.keras.layers.Dense(1, activation='sigmoid')
        self.cvr_tower = tf.keras.layers.Dense(1, activation='sigmoid')
    
    def call(self, inputs):
        feat = self.embedding(inputs['item_id'])
        return {
            'ctr_prob': self.ctr_tower(feat),
            'cvr_prob': self.cvr_tower(feat)
        }
```

### 不确定性加权Loss

```python
class UncertaintyWeightedLoss(nn.Module):
    def __init__(self, num_tasks):
        super().__init__()
        self.log_sigma = nn.Parameter(torch.zeros(num_tasks))
    
    def forward(self, losses):
        precision = torch.exp(-self.log_sigma)
        return 0.5 * sum(p * l for p, l in zip(precision, losses)) + torch.sum(self.log_sigma)
```

### DPP贪心采样

```python
def dpp_greedy(L, k):
    """L: 核矩阵, k: 选取数量"""
    selected = []
    for _ in range(k):
        gains = []
        for i in range(len(L)):
            if i not in selected:
                det = np.linalg.det(L[selected + [i]][:, selected + [i]])
                gains.append((det, i))
        best = max(gains)[1]
        selected.append(best)
    return selected
```
