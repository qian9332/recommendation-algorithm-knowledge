### 涵盖内容

| 模块 | 核心内容 |
|-----|---------|
| **深度召回技术** | DSSM损失函数推导、MIND多兴趣召回、SDM序列召回 |
| **高级排序模型** | DCN-V2交叉网络推导、行为序列Attention机制 |
| **因果推断与去偏** | 位置偏差消除、选择偏差消除、IPS/DR方法 |
| **序列推荐** | SASRec、BERT4Rec、TiSASRec |
| **图神经网络推荐** | LightGCN原理与实现 |
| **对比学习推荐** | SGL、SimGCL、NCL |
| **强化学习推荐** | DQN、PPO、Actor-Critic |
| **大模型与推荐** | LLM特征增强、生成式推荐 |
