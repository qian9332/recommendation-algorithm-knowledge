## 📚 知识体系概览

本文档系统梳理了搜索、广告、推荐（搜广推）系统的核心算法与架构知识，涵盖从召回到排序、从特征工程到模型部署的全链路技术栈。

### 🎯 核心模块

| 模块 | 关键技术 | 工业级指标 |
|-----|---------|-----------|
| **召回** | 多路召回、倒排索引、双塔DSSM | 召回率65%+，延迟<15ms |
| **排序** | 特征工程、交叉特征、多模态融合 | CTR+3%，AUC+0.5% |
| **多任务** | MMoE/PLE、GradNorm、ESMM | GMV+3.4%，跷跷板缓解 |
| **重排** | DPP多样性、RL重排、疲劳建模 | 多样性+8%，留存+1.2% |
| **系统** | 模型分片、实时特征、A/B实验 | P99<30ms，QPS 5w+ |

                ↑                              ↓

### 多任务学习网络对比
```
MMoE:  共享Expert → Gate → Tower (易跷跷板)

```

## 🔑 核心知识点速查

### 召回层关键参数

| 参数 | 推荐值 | 说明 |
|-----|--------|------|
| 召回候选数 | 500-2000 | 平衡效果与延迟 |
| 向量维度 | 64-128 | 平衡精度与内存 |
| 负采样比例 | 1:4~1:10 | 正负样本平衡 |

### 排序层关键参数

| 参数 | 推荐值 | 说明 |
|-----|--------|------|
| Embedding维度 | 32-128 | 根据数据量调整 |
| 隐藏层维度 | 256-1024 | 过大收益递减 |
| 学习率 | 1e-4 ~ 1e-3 | Adam优化器 |

### 系统性能阈值

| 指标 | 绿色 | 黄色 | 红色 |
|-----|------|------|------|
| P99延迟 | <30ms | 30-50ms | >50ms |
| QPS下降 | <5% | 5-10% | >10% |
| 错误率 | <0.1% | 0.1-1% | >1% |

更多参数请查看 [快速参考卡片](docs/12-quick-reference.md)

## 📈 效果指标速查

### 召回层指标

| 指标 | 计算方式 | 目标值 |
|-----|---------|--------|
| Recall@K | 用户转化集合被召回的比例 | >65% |
| Precision@K | 召回结果中有效商品比例 | >8% |
| 覆盖率 | 被召回商品占全库比例 | >30% |
